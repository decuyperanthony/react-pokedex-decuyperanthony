<!-- GETTING STARTED -->
## Getting Started


### Installation

1. Clone the repo
```sh
git clone https://github.com/decuyperanthony/react-pokedex-decuyperanthony
```

2. Install YARN client packages && start server
```sh
yarn
yarn start
```
